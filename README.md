GEEK-IN — Treino Nerd

Site oficial da Geek-In, uma marca que une cultura nerd + treino pesado.

A ideia da marca nasceu de uma pergunta simples:

Se eu gosto de treinar e gosto de nerdices, por que não juntar os dois?

A Geek-In cria produtos para quem gosta de treinar com identidade, usando camisetas, acessórios e itens inspirados em cultura geek.

Conceito da Marca

A Geek-In não é só roupa de treino.

Ela representa pessoas que:

treinam pesado

gostam de cultura nerd

querem se expressar através do que vestem

Aqui você encontra camisetas que misturam:

-games
-fantasia
-ficção científica
-academia

com conforto real para treinar.
